# Dharavahik

[Dharavahik](https://dharavahik.vercel.app) is a web application where you can easily watch any movie you want. It is built using Next.js and hosted on Vercel.

## Features

- Browse and search for movies.
- Watch movies with a user-friendly interface.
- Responsive design for mobile and desktop.

## Getting Started

Follow these instructions to set up the project locally on your machine.

### Prerequisites

- Node.js installed (version 14.x or above).
- npm (comes with Node.js).

### Installation

#### 1. Clone the Repository

```bash
git clone https://github.com/yourusername/dharavahik.git
cd dharavahik
```

#### 2. Install Dependencies
```bash
npm install
```

### 3. Set Up Environment Variables
- Rename .env.example to .env.
- Update the .env file with your API keys and other necessary configurations.

### 4. Run the Development Server
```bash
npm run dev
```
